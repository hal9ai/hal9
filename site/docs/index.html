hal9: Interactive Data Apps without Web Dev

Hal9 helps you build web applications with not web development background to build rich web applications from Python and R. This project is under development in Hal9â€™s GitHub repo.

Easy to Use

  • R
  • Python
devtools::install_github("hal9ai/hal9ai", subdir = "r")
library(hal9)

h9_create() |>
  h9_load(iris) |>
  h9_dropdown(values = "versicolor,setosa,virginica") |>
  h9_filter(field = "Species", expression = "field == outputs.dropdown") |>
  h9_scatter_chart(x = "Sepal.Length", y = "Sepal.Width", color = "Species")
